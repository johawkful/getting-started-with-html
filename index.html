Essential Information

HTML

HyperText Markup language (HTML) is the stanard language used to create web pages. There are many different parts to HTML: text content, markup, references to other documents, and links. The text is the content of the web page. The markup is the design of the web page. References to other documents can be images or videos on the web page. The final part of HTML would be links that take you to other web pages.

Major Pieces of the Web

The user is an essential part of the internet. It is the user that types in the url to bring up a web page. The user uses their computer to establish a connection with their browser. The browser communicates with the server to bring up the content on a webpage.

Elements, Tags, and Attributes

All HTML documents are made up of elements. The computer reads the different elements by looking at the tags next to the text, video, or pictures. Attributes are used to create links and put images on a page.

Inline vs. Block

All elements in HTML are either block or inline. Block elements create a box around the content that can be manipulated.

Tags

Most of the tags must have opening tags and closing tags. Some of these are void tags, which means they do not require ending tags.

  • b-bold
  • em-italics
  • a href="url"- links
  • img src="url" alt="text"- images(void tag)
  • p-paragraph
